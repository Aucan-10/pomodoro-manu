
# Pomorolo

## Descripción

Pomorolo es un proyecto en desarrollo que busca implementar una solución innovadora para la gestión y optimización de tareas mediante la técnica Pomodoro.

## Características (Próximamente)

- Temporizador Pomodoro configurable
- Opcion para detenerlo por 5 o 10 minutos
- Opcion para detenerlo

## Estado del Proyecto

🚧 **En construcción** - Este proyecto se encuentra en fase de desarrollo activo.

## Requisitos

- Node.js (versión recomendada)
- npm o yarn

## Instalación

```bash
git clone <url-del-repositorio>
cd pomorolo
npm install
```

## Uso

```bash
npm start
```
